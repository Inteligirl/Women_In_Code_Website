Learn to Code

On my journey to learn computer programming I came across some really great websites for people wanting to learn on thier own and at their own pace. Some of these sites are specific to certain development topics and the table below contains links to learning websites. I'll include the webiste along with a brief description of its contents.

For further reading

This is an unordered list of website that I find useful for programming and development. There is a vast list of websites available to learn more about this topic, however I chose a few of my favorties to share here.

  • Eloquent Javascript - A modern introduction to programming. This is a recommended book for those interested in learning Javascript.
  • Cplusplus.com - One of my favortie reference websites for questions regarding the programming language C++. Great forum support and tutorials.
  • Mozilla Development Network - A website hosted by Mozilla to learn web development for beginners.
  • CSS Tricks - A website devoted to CSS and has many tutorials on various ways to maximize use of CSS. CSS Tricks also have a video and podcast with tutorials.
  • Google Fonts - A great resource for fonts to include on websites with a link.
  • Michael Crump - A tutorial for setting up Git with Visual Studio to push your work to github.
  • Codepen.io - A free website to host HTML, CSS and Javascript for the public to view. You can fork other projects into your account and work with the code as well.
  • Smashing Magazine - Online magazine for web developers. Up to date articles on trends in website technologies.
  • Really Good Emails - A compilation of email templates that use a wide range of web technologies.
